We Made a Discord Channel

We made a channel called Code, Hack, and Co, where we could talk about subjects forbidden in the MLH server where we met. We also thought that it would be a good place to talk and share our screens within a more secluded voice channel.

In our channel we discovered that we share a love of IOT devices and decided that we would both like to work through the book Linux Basics for Hackers by OccupyTheWeb.

We Made a Website

Malik made a website to help people in the MLH Discord server. He used HTML and asked people to help create the styling for it as an open source project for Global Hack Week. With a helpful hand from Alberto and VRX, we used Git and GitHub to collaborate on the project.

We Made Discord Bots Together

We decided to learn how to make Discord bots together. We watched this YouTube video and cobbled together a bot that could respond to a few commands. We learned how to use the Discord API and how to use the Python programming language to make a bot that could respond to commands. We learned how to use the discord.py library to make the bot. After watching the YouTube video, this was a good resource. The next task would be to learn how to make the bot run on a server so that it could be used by others.
